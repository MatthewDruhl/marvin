# MoRyde AllTrak Qualifier Measurements

Source: [MoRyde AllTrak Qualifier Video](https://www.youtube.com/watch?v=ijUPxecw-yI)

Measurements needed to determine if your trailer's suspension is compatible with the AllTrak equalizer suspension system.

## Key Concepts

- **Overslung axle:** Leaf spring is above the axle
- **Underslung axle:** Leaf spring is below the axle
- Leaf spring lengths are listed at full/design load (shorter when unloaded)
- Measure hanger spread edge-to-edge if hangers are straight and same width

## Measurements

| Step | What to Measure | Requirement |
|------|----------------|-------------|
| **1** | Center hanger height (frame bottom to center of hole) | **3.25" min** (overslung) or **4.25" min** (underslung) |
| **2** | Frame hanger width | **3" wide**, hole centered **1.5"** from edges |
| **3** | Height adjustment holes (if present) | Must use lower hole, or cut off bottom material if using upper hole |
| **4** | Leaf spring width | **1.75"** |
| **5** | Frame hanger spread (front to rear hanger) | 31" wheelbase: **55"**, 33" wheelbase: **59"**, 35" wheelbase: **61"** |
| **6** | Shackle link length | **2.25"** standard (3.125" only with overslung + 4.25" hanger) |
| **7** | Axle type | **Straight axle only** (no drop axles) |
| **8** | Rim size | **15" min** (overslung) or **16" min** (underslung) |
| **9A** | Hanger face to tire sidewall clearance | **1-11/16" min** |
| **9B** | Top of tire to underside of floor | **3" min** (measure from lowest obstruction) |
| **9C** | Underside of frame to axle assembly | **2.5" min** |
| **9D** | Equalizer height (bottom of bolts) | Record measurement |

## Tolerances (Step 5)

- 31" and 33" wheelbase: 1/8" under allowed
- 35" wheelbase: 1/4" under allowed (more forgiving due to extra tire spacing)

## Notes

- 31" wheelbase only used with 24" leaf springs
- Drop axles (with a jog/arm) cannot be used, only straight axles
- Step 9B: Watch for outriggers, patio tube structures, or anything hanging low between tire and floor
