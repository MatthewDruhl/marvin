## Behavioral Questions (STAR Format)

### General
1. Tell me about yourself and your background
2. Describe a challenging technical problem you solved
3. Tell me about a time you disagreed with a team member
4. Describe a project you're most proud of
5. How do you handle tight deadlines?
6. Tell me about a time you had to learn a new technology quickly
7. Describe a situation where you had to debug a critical production issue
8. How do you prioritize when you have multiple competing tasks?

### Leadership & Collaboration
1. Describe a time you mentored a junior developer
2. Tell me about a time you led a project or initiative
3. How do you handle code reviews — giving and receiving feedback?

## Technical Questions

### Python
1. Explain the GIL and its implications
2. Difference between list, tuple, set, and dict
3. How does Python's memory management work?
4. Explain decorators and give a use case
5. What are generators and when would you use them?
6. Explain async/await in Python

### Data Structures & Algorithms
1. Implement a binary search
2. Explain Big O notation with examples
3. When would you use a hash map vs a tree?
4. Implement a queue using two stacks
5. Explain the difference between BFS and DFS
6. Sort algorithm trade-offs (quick sort vs merge sort vs heap sort)

### System Design
1. Design a URL shortener
2. Design a job queue system
3. How would you design a REST API for a social media app?
4. Explain microservices vs monolith trade-offs
5. How would you handle database scaling?

### Backend / Infrastructure
1. Explain REST vs GraphQL
2. How do you handle database migrations?
3. Explain Docker containers and orchestration
4. How do you approach API versioning?
5. Describe your testing strategy (unit, integration, e2e)
